## 文件管理方式

如果有D盘，则你的D盘应当有如下路径

|            .temp路径            |               Ovalene文件夹               |         development文件夹         | software文件夹 |
| :------------------------------: | :----------------------------------------: | :-------------------------------: | :------------: |
| 用于存放某些软件<br />的临时文件 |           用于管理某些文档和文件           |        用于存放代码和环境        |     放软件     |
|                                  | mass,dow,video,picture<br />都应该放在这里 | 下分requirement<br />和code文件夹 |                |
